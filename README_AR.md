# ProDesign - موقع احترافي حديث

موقع ويب احترافي وحديث مصمم باستخدام React مع تصميم جميل وتجربة مستخدم عالية.

## ✨ الميزات الرئيسية

### 1. **التصميم الحديث والاحترافي**
- واجهة مستخدم جميلة وعصرية
- ألوان متناسقة وتدرجات لونية احترافية
- رسوم توضيحية وعناصر بصرية جذابة

### 2. **تجربة مستخدم متقدمة**
- تخطيط سلس وسهل الاستخدام
- رموز emoji جذابة في كل قسم
- أنيميشنات سلسة وسحرية
- الموقع متوافق 100% مع الأجهزة المختلفة

### 3. **المكونات الرئيسية**

#### ✅ Header (رأس الصفحة)
- قائمة تنقل ملتصقة في الأعلى
- شعار احترافي بتدرج لوني
- قائمة هاتف ذكي متجاوبة
- تأثيرات حركة عند المرور على الروابط

#### 🎯 Hero Section (القسم الرئيسي)
- عنوان قوي وجذاب
- نص فرعي واضح
- زر دعوة للعمل (CTA)
- رسوميات متحركة جميلة

#### 🛠️ Services (الخدمات)
- عرض 6 خدمات مختلفة
- تصميم بطاقات جميل
- أيقونات emoji ديناميكية
- تأثيرات رفع عند الوقوف على البطاقة

#### 🎨 Portfolio (أعمالنا)
- عرض شبكي للمشاريع
- رسوميات توضيحية لكل مشروع
- تأثيرات تفاعلية عند الوقوف على المشروع
- تصميم احترافي ومنظم

#### 📊 About (من نحن)
- معلومات عن الشركة
- إحصائيات مهمة (المشاريع، العملاء، الفريق)
- رسوميات توضيحية جميلة
- تصميم نصف صفحة مع صفحة أخرى للصورة

#### 📧 Contact (التواصل)
- نموذج اتصال كامل
- معلومات التواصل (العنوان، الهاتف، البريد)
- تصميم احترافي وسهل الاستخدام
- رسالة شكر عند الإرسال

#### 🔗 Footer (تذييل الصفحة)
- روابط سريعة للأقسام
- روابط التواصل الاجتماعي
- حقوق النشر

## ⚡ التكنولوجيا المستخدمة

- **React 19.2.4** - مكتبة JavaScript للواجهات
- **CSS3** - تنسيق حديث مع:
  - CSS Grid و Flexbox
  - Gradients و Transitions
  - Animations الديناميكية
  - Media Queries للتجاوب

## 📱 التجاوب (Responsive Design)

الموقع متوافق تماماً مع:
- ✅ الحواسيب المكتبية
- ✅ الأجهزة اللوحية

## 💡 نصائح التخصيص

### تغيير الألوان
عدّل متغيرات CSS في `styles/global.css`:
```css
:root {
  --primary-color: #6366f1;      /* الأزرق الأساسي */
  --secondary-color: #ec4899;    /* الوردي الثانوي */
  /* ... */
}
```

### تغيير المحتوى
- عدّل نصوص المكونات في ملفات `.js`
- غيّر الخدمات والمشاريع حسب احتياجاتك
- أضف صورك الخاصة بدل الرموز التعبيرية

### إضافة صفحات جديدة
1. أنشئ مكون جديد في `src/components/`
2. أنشئ ملف CSS له في `src/styles/`
3. استورده في `App.js`
